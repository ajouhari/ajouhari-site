General algorithms

There's a whole host of simple tasks that can be done more efficiently by algorithms, saving people much time and effort. Here I have documented a wide array of algorithms, of varying complexity and purposes.

Most of the algorithms displayed are in Python; however, moving forward, many of the algorithms will be written in C++ as it has a runtime advantage. Keeping best practices in mind, I always aim to write my code to be simple, well-documented, and with the fastest runtime possible.

Search algorithms

Search algorithms have two primary function: to find if a path exists between two locations, and if so, to find the shortest path between them. Here I will code and review multiple search algorithms, from a brute-force method, to a well-designed a-star search algorithm.

As trivial as they may seem looking from the outside, the inner working of a search algorithm is a complex and highly important mechanism of the internet. I try my best to understand and then explain the multiple ways to implement it and the advantages that each may serve.

Sort algorithms

From databases to e-commerce sites, sorting engines play a cruical role that makes a substancial difference. There are many ways to go about making one, and with today’s big data, it becomes increasingly important to use more and more efficient sorting algorithms to sift through the data.

Here I will code and review some of the best known sorting algorithms. For the purpose of simplicity and explaination, it will be written in Python, which is a great language to learn coding with.
